# SEO Optimization Guide for PesoWise

## Overview
This document outlines the comprehensive SEO strategy implemented for PesoWise by Unimisk, focusing on search engine optimization and Generative Engine Optimization (GEO) for AI-powered search engines.

## Target Keywords

### Primary Keywords
- **pesowise**
- **pesowise unimisk**
- **petty cash management**
- **expense tracking software**
- **expense management software**

### Secondary Keywords
- employee reimbursement software
- financial workflow automation
- SaaS expense platform

- cash flow management software

- business expense software

## SEO Implementation

### 1. Meta Tags (index.html)
- **Title**: Optimized with primary keywords
- **Description**: Comprehensive description with keywords
- **Keywords**: Extensive keyword list including all target terms
- **Open Graph**: Social media optimization
- **Twitter Cards**: Twitter-specific optimization
- **Structured Data**: JSON-LD schema markup

### 2. Generative Engine Optimization (GEO)
Special meta tags added for AI crawlers:
- `ai:description` - Detailed description for AI understanding
- `ai:keywords` - Keyword-rich content for AI indexing
- `ai:category` - Category classification
- `ai:use-case` - Use case descriptions

### 3. Robots.txt Configuration
- Allows all major search engines
- Allows AI crawlers (GPTBot, ChatGPT-User, CCBot, Claude-Web, PerplexityBot)
- Blocks private/admin routes
- Optimized crawl delays

### 4. Sitemap.xml
- Includes public pages only
- Proper priority and change frequency
- Image sitemap support
- Regular updates

### 5. Public Landing Page
A comprehensive landing page (`/` and `/home`) that:
- Contains keyword-rich content
- Provides clear value proposition
- Includes structured content for crawlers
- Optimized for both users and search engines

## Public Pages for SEO

### Accessible Pages
1. **Homepage** (`/`) - Public landing page
2. **Landing Page** (`/home`) - Alternative landing page
3. **Auth Page** (`/auth`) - Login/signup page

### Protected Pages (Not Indexed)
- `/dashboard` - User dashboard
- `/expenses` - Expense management
- `/admin/*` - Admin panels
- `/settings` - User settings
